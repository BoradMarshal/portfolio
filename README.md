# Marshal Borad Portfolio

This is a premium, cinematic portfolio designed for a UI/UX Designer & Frontend Developer.

## How to Go Live (Get a Link)

### Option 1: Netlify Drop (Easiest & Fastest)
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop this entire `portfoliyo` folder onto the browser window.
3. Within seconds, you will get a live link (e.g., `https://creative-marshal.netlify.app`).

### Option 2: GitHub Pages (Professional)
1. Create a new repository on [GitHub](https://github.com/new).
2. Upload these files to the repository.
3. Go to **Settings > Pages** and select the `main` branch as the source.
4. Your link will be `https://your-username.github.io/repository-name`.
